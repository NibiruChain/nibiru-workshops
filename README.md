# nibiru-workshops

Public technical workshops for Nibiru Chain.


## Getting Started

Install `just` to run project-specific commands.

```bash
# You'll need Rust installed for `cargo` commands.
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

```bash
cargo install just
```
